# Танцевальные таблицы — PNG конвертер

Загружаешь PNG с таблицей танцевальных шагов (английский) → получаешь красивую переведённую PNG.

## Запуск

1. Получи бесплатный ключ Gemini API: https://aistudio.google.com/apikey
2. Скопируй `.env.example` в `.env` и вставь ключ:
   ```
   cp .env.example .env
   ```
3. Установи зависимости (если ещё нет):
   ```
   npm install
   ```
4. Запусти сервер:
   ```
   node server.js
   ```
5. Открой http://localhost:3000
