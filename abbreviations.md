# Расшифровка сокращений

Справочник сокращений, используемых в таблицах танцевальных фигур (английский оригинал → русский перевод в системе).

## Ритм

| Англ. | Рус. | Расшифровка |
|-------|------|-------------|
| S / SLOW | М | Медленно (Slow) |
| Q / QUICK | Б | Быстро (Quick) |

**Ритм-паттерны (колонка Итого):**

| Паттерн | Рус. | Описание |
|---------|------|----------|
| SS | ММ | Два медленных |
| QQ | ББ | Два быстрых |
| QQS | ББМ | Два быстрых + медленный |
| SQQ | МББ | Медленный + два быстрых |
| SSQQ | ММББ | Два медленных + два быстрых |
| SSQQ&S | ММББ&М | Два медленных + два быстрых + «и» + медленный |

## Танго — специфика

В Танго колонка **COUNT** содержит S/Q, которые транслируются:
- `S` → `М` (медленно)
- `Q` → `Б` (быстро)

Колонка **BEATS (8CT)** — числовой счёт в восьмитактовой структуре: `1,2` / `3,4` / `5` / `6` / `7,8` — копируется как есть.
